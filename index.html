Getting Started

Pupeteer is a scroll aware animation library for creating websites. It was created for personal use by myself. Getting started with pupeteer is fairly simple - you just need to reference the JavaScript file (which uses ES5 for compatibility) and annotate your html elements with special attributes, such as p-animation.
